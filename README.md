
# Tinier Databses
Tinier is a URL Shortener REST API written in go.

## Installation
```
$ git clone https://github.com/pooladkhay/tinier-dbs.git
$ cd tinier-dbs/cassandra
$ docker-compose up --build -d
$ cd ../redis
$ docker-compose up --build -d
```
